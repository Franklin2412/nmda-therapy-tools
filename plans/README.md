## üéØ Project Goals

### Primary Objective
Create **free, accessible, web-based therapeutic tools** that help children recovering from NMDA encephalitis rebuild:
- **Cognitive skills** (memory, attention, problem-solving)
- **Speech and language abilities**
- **Emotional regulation**
- **Daily living skills**
- **Physical coordination**

### Key Principles
1. **Evidence-Based**: Grounded in clinical research on encephalitis rehabilitation
2. **Child-Friendly**: Engaging, colorful, game-based interfaces
3. **Accessible**: Works for children with varying abilities; keyboard navigable, screen-reader compatible
4. **Privacy-Focused**: All data stored locally, no servers, no tracking
5. **Free Forever**: Open-source, hosted on GitHub Pages
6. **Medical Integration**: Developed with input from healthcare professionals
